# Projeto 3: Snaze

* **Como compilar e executar projeto:**

    - Observação: 
        1. O jogo precisa ser executado dentro da raíz do diretório.

    - No Linux:
        1. g++ src/*.cpp -o main -I./include
        2. ./main
    - No Windows:
        1. g++ src/*.cpp -o main -I./include
        2. .\main.exe
